\hypertarget{index_intro_sec}{}\doxysection{Введение}\label{index_intro_sec}
Этот проект состоит из пользовательского TCP сервера, реализованного с помощью Qt. Сервер способен работать с несколькими клиентами, обрабатывать входящие сообщения и вести статистику пользователей в базе данных SQLite.\hypertarget{index_server_sec}{}\doxysection{Сервер}\label{index_server_sec}
Сервер реализован в классе \mbox{\hyperlink{class_my_tcp_server}{My\+Tcp\+Server}}. Он прослушивает входящие соединения, обрабатывает чтение и запись данных от/к клиентам и управляет отключениями клиентов.\hypertarget{index_database_sec}{}\doxysection{База данных}\label{index_database_sec}
Базой данных SQLite управляет синглтон класса \mbox{\hyperlink{class_database}{Database}}. Он включает в себя функции инициализации базы данных, управления данными пользователей и запроса статистики пользователей.\hypertarget{index_functions_sec}{}\doxysection{Функции}\label{index_functions_sec}
Заголовочный файл \mbox{\hyperlink{functions_8h}{functions.\+h}} содержит несколько вспомогательных функций, используемых во всем проекте. Они включают разбор входящих данных клиента, обработку авторизации и регистрации пользователей, обновление статистики пользователей и управление дескрипторами сокетов пользователей.\hypertarget{index_tcp_sec}{}\doxysection{TCP}\label{index_tcp_sec}
TCP связь между сервером и клиентами обрабатывается с помощью классов QTcp\+Server и QTcp\+Socket. Данные отправляются и принимаются в формате QByte\+Array.

В этой документации содержится подробная информация о каждом компоненте проекта, используемых структурах данных и о том, как компоненты взаимодействуют друг с другом.
